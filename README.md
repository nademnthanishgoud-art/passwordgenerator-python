# Python Password Generator

## Description
A simple Python Password Generator that creates secure passwords using random letters, numbers, and special characters based on user input.

## Features
- Generates random passwords
- User chooses the number of letters, numbers, and symbols
- Uses Python's built-in `random` module

## Technologies Used
- Python 3

## How to Run
1. Run `password generator.py`
2. Enter the required number of letters, numbers, and symbols.
3. The program generates a random password.
